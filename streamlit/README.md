# 🀄 麻雀成績ダッシュボード

こちらはClaude Codeを用いて作成したものです。  
Streamlit + Plotly で作られた**麻雀成績の可視化アプリ**です。  
CSVファイルに記録された対局データを読み込み、インタラクティブなグラフで成績を分析できます。

---

## 📂 ファイル構成

```
streamlit/
├── app.py              # ホームページ（アプリのエントリーポイント）
├── utils.py            # データ読み込み・前処理の共通関数
├── requirements.txt    # 必要なPythonライブラリ一覧
├── res_input.csv       # Hamada の個人成績データ（フリー・セット）
├── set_res.csv         # セット対局の全プレイヤー成績データ
└── pages/
    ├── 1_🀄_Hamada.py  # ページ1：Hamada の個人成績ダッシュボード
    └── 2_👥_Players.py # ページ2：プレイヤー別成績比較
```

---

## 🚀 起動方法

```bash
# 必要なライブラリをインストール
pip install -r requirements.txt

# アプリを起動
streamlit run app.py
```

起動後、ブラウザが自動で開きます（通常 `http://localhost:8501`）。

## 📄 ページ一覧

### 🏠 ホーム（`app.py`）

アプリのトップページです。サイドバーのメニューから各ページに移動できます。

---

### 🀄 ページ1：Hamada の成績（`pages/1_🀄_Hamada.py`）

**データソース：** `res_input.csv`（Hamada の全対局記録）

Hamada さん個人の成績を多角的に分析するページです。

#### 🔧 サイドバーフィルター

| フィルター | 説明 |
|---|---|
| **対局種別** | フリー（F）・セット（S）・全てから選択 |
| **グループ** | セット対局の場合、グループを絞り込み可能 |
| **日付範囲** | 分析対象の期間を指定 |

#### 📊 表示される可視化・指標

| セクション | 内容 |
|---|---|
| **KPI メトリクス（6つ）** | 対局数・累計スコア・1局平均スコア・トップ率・ラス率・平均順位を大きく表示 |
| **📈 累計スコア推移** | 全て・フリー・セットを切り替えて表示できる折れ線グラフ。対局を追うごとにスコアがどう変化したかを確認 |
| **🏆 順位分布** | 1位〜4位それぞれの回数と割合を棒グラフで表示 |
| **📊 グループ別平均スコア** | グループまたは対局種別ごとの平均スコアを棒グラフで比較（プラスは緑、マイナスは赤） |
| **🎯 局番ごとの平均順位・平均スコア** | 1局目・2局目など局番ごとの成績を棒グラフで確認。得意な局番・苦手な局番がわかる |
| **📅 月別スコア推移** | 月ごとの合計スコアを棒グラフで表示。調子の良かった月・悪かった月を把握できる |
| **🎲 順位別スコア分布（Box plot）** | 1位〜4位それぞれのスコアのばらつきを箱ひげ図で表示。中央値・平均値・標準偏差が一目でわかる |
| **🕐 直近20局の順位推移** | 最新20局の順位を棒グラフで表示。連続トップ・連続ラスの自動検出あり。右側に直近のKPIも表示 |
| **📉 スコア分布（ヒストグラム）** | 1局ごとのスコアの分布をヒストグラムで表示。フリー・セットで色分けされる |
| **📅 セッション別成績** | 日付ごとの対局数・合計スコア・平均順位などをテーブル形式で表示（新しい日付順） |
| **📋 詳細データ（展開可能）** | 全対局の生データをテーブルで確認できる |

### 👥 ページ2：プレイヤー別成績（`pages/2_👥_Players.py`）

**データソース：** `set_res.csv`（セット対局の全プレイヤーデータ）

グループ・プレイヤーを選択して、特定プレイヤーの成績を分析するページです。

#### 🔧 サイドバーフィルター

| フィルター | 説明 |
|---|---|
| **グループ** | 参加グループを選択 |
| **プレイヤー** | 選択したグループ内のプレイヤーを選択 |
| **日付範囲** | 分析対象の期間を指定 |

#### 📊 表示される可視化・指標

| セクション | 内容 |
|---|---|
| **📊 グループ通算ポイントランキング（テーブル）** | グループ全員の対局数・合計スコア・平均スコア・トップ率・平均順位を一覧表示。選択中のプレイヤーはハイライト表示 |
| **通算ポイントランキング（棒グラフ）** | グループ内全プレイヤーの累計スコアを横棒グラフで比較（プラスは緑、マイナスは赤） |
| **KPI メトリクス（6つ）** | 選択プレイヤーの対局数・累計スコア・1局平均・トップ率・ラス率・平均順位を大きく表示 |
| **📈 累計スコア推移** | 選択プレイヤーの累計スコアの折れ線グラフ。ゼロ基準線からの面積も塗りつぶして表示 |
| **🏆 順位分布** | 1位〜4位の回数と割合を棒グラフで表示 |
| **📅 月別スコア推移** | 月ごとの合計スコアを棒グラフで表示 |
| **🎲 順位別スコア分布（Box plot）** | 各順位でのスコアのばらつきを箱ひげ図で表示 |
| **🕐 直近20局の順位推移** | 最新20局の順位推移を棒グラフで表示。連続トップ・連続ラスの自動検出あり |
| **📉 スコア分布（ヒストグラム）** | スコアの分布をヒストグラムで表示。平均値を縦線でマーク |
| **📅 セッション別成績（テーブル）** | 日付ごとの集計をテーブルで表示 |
| **📋 全対局データ（展開可能）** | 全対局の生データをテーブルで確認できる |

## 📋 データファイルについて

### `res_input.csv`（Hamada 個人成績）

| 列名 | 意味 |
|---|---|
| `num` | 通しの対局番号 |
| `date` | 対局日（`YYYY/MM/DD` 形式） |
| `match` | 局番（例：1局目、2局目） |
| `ForS` | 対局種別（`F`=フリー、`S`=セット） |
| `member` | グループ名（セット対局の場合） |
| `rank` | 順位（1〜4） |
| `score` | 生点（素点） |
| `pm` | ±スコア（ウマ・オカ込みの最終得点） |
| `cumsum` | 累計スコア |

### `set_res.csv`（セット対局 全プレイヤー）

対局ごとに1〜4位のプレイヤー名とスコアを横持ちで記録したファイルです。  
アプリ内で自動的に縦持ち（1行＝1プレイヤー）形式に変換して使用されます。

## 💡 操作のポイント

- **グラフはすべてインタラクティブ**です。マウスオーバーで詳細値が表示され、凡例クリックで表示/非表示の切り替えができます。
- サイドバーのフィルターを変更すると、**ページ内のすべてのグラフが自動で更新**されます。
- 「📋 詳細データ」などの折りたたまれたセクションは、クリックで展開できます。
